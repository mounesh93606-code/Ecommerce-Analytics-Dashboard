# E-Commerce Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed using the Superstore dataset. The dashboard provides insights into sales performance, profitability, quantity sold, and regional trends.

## Tools Used
- Power BI
- Superstore Dataset
- Data Visualization

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales by Category
- Profit by Category
- Sales Trend Over Time
- Region Filter
- State-wise Sales Treemap

## Key Insights
- Technology category generated the highest sales.
- Office Supplies and Furniture contributed significantly to revenue.
- Sales performance can be analyzed by state and region.
- Interactive filters allow detailed exploration of data.
